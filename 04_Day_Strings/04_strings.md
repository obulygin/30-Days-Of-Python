# –î–µ–Ω—å 4

## –°—Ç—Ä–æ–∫–∏

–¢–µ–∫—Å —è–≤–ª—è–µ—Ç—Å—è —Ç–∏–ø–æ–º –¥–∞–Ω–Ω—ã—Ö "—Å—Ç—Ä–æ–∫–∞"(string). –õ—é–±–æ–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö, –∑–∞–ø–∏—Å–∞–Ω–Ω—ã–π –≤ –≤–∏–¥–µ —Ç–µ–∫—Å—Ç–∞, —è–≤–ª—è–µ—Ç—Å—è —Å—Ç—Ä–æ–∫–æ–π. –õ—é–±—ã–µ –¥–∞–Ω–Ω—ã–µ, –∑–∞–∫–ª—é—á–µ–Ω–Ω—ã–µ –≤ –æ–¥–∏–Ω–∞—Ä–Ω—ã–µ, –¥–≤–æ–π–Ω—ã–µ –∏–ª–∏ —Ç—Ä–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏, —è–≤–ª—è—é—Ç—Å—è —Å—Ç—Ä–æ–∫–∞–º–∏. –í Python —Å—É—â–µ—Å—Ç–≤—É—é—Ç —Ä–∞–∑–ª–∏—á–Ω—ã–µ –º–µ—Ç–æ–¥—ã —Å—Ç—Ä–æ–∫ –∏ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å –¥–∞–Ω–Ω—ã–º–∏ —Ç–∏–ø–∞ "—Å—Ç—Ä–æ–∫–∞". –ß—Ç–æ–±—ã –ø—Ä–æ–≤–µ—Ä–∏—Ç—å –¥–ª–∏–Ω—É —Å—Ç—Ä–æ–∫–∏, –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ –º–µ—Ç–æ–¥ len().

### –°–æ–∑–¥–∞–Ω–∏–µ —Å—Ç—Ä–æ–∫


### –ö–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü–∏—è —Å—Ç—Ä–æ–∫

–ú—ã –º–æ–∂–µ–º –æ–±—ä–µ–¥–∏–Ω—è—Ç—å —Å—Ç—Ä–æ–∫–∏. –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ –∏–ª–∏ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –∫–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü–∏–µ–π.

–ü—Ä–∏–º–µ—Ä:

```py
first_name = '–ü–∏—Ç–µ—Ä'

space = ' '

print(full_name) # –ü–∏—Ç–µ—Ä –ü–∞—Ä–∫–µ—Ä
# –ü—Ä–æ–≤–µ—Ä–∫–∞ –¥–ª–∏–Ω–Ω—ã —Å—Ç—Ä–æ–∫–∏ —Å –ø–æ–º–æ—â—å—é –≤—Å—Ç—Ä–æ–µ–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ len()
print(len(first_name))  # 5
print(len(last_name))   # 6
print(len(last_name) > len(first_name)) # True
print(len(full_name)) # 12
```

### –≠–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–µ–π

–î–∞–≤–∞–π—Ç–µ –≤–∑–≥–ª—è–Ω–µ–º –Ω–∞ –Ω–∞–∏–±–æ–ª–µ–µ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–Ω—ã–µ —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ —Å–∏–º–≤–æ–ª—ã:

- \n: –ü–µ—Ä–µ—Ö–æ–¥ –Ω–∞ –Ω–æ–≤—É—é —Å—Ç—Ä–æ–∫—É
- \t: –ì–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω–∞—è —Ç–∞–±—É–ª—è—Ü–∏—è (2-4 –ø—Ä–æ–±–µ–ª–∞)
- \\\\: –û–±—Ä–∞—Ç–Ω–∞—è –∫–æ—Å–∞—è —á–µ—Ä—Ç–∞
- \\': –û–¥–∏–Ω–∞—Ä–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏ (')
- \\": –î–≤–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏ (")

```py
print('–Ø –Ω–∞–¥–µ—é—Å—å, —á—Ç–æ –∫–∞–∂–¥—ã–π –ø–æ–ª—É—á–∞–µ—Ç —É–¥–æ–≤–æ–ª—å—Å—Ç–≤–∏–µ –æ—Ç Python Challenge.\n–ê –≤—ã –ø–æ–ª—É—á–∞–µ—Ç–µ?') # –ü–µ—Ä–µ–Ω–æ—Å —Å—Ç—Ä–æ–∫–∏
```

### –§–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ —Å—Ç—Ä–æ–∫

#### –°—Ç–∞—Ä—ã–π —Å—Ç–∏–ª—å —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è —Å—Ç—Ä–æ–∫ (% –û–ø–µ—Ä–∞—Ç–æ—Ä)

–í Python —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Å–ø–æ—Å–æ–±–æ–≤ —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è —Å—Ç—Ä–æ–∫. –°–µ–π—á–∞—Å –º—ã —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –∏–∑ –Ω–∏—Ö. –û–ø–µ—Ä–∞—Ç–æ—Ä "%" –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è –Ω–∞–±–æ—Ä–∞ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö –æ–±—ä–µ–¥–∏–Ω–µ–Ω–Ω—ã—Ö –≤ –∫–æ—Ä—Ç–µ–∂ (tuple), –≤–º–µ—Å—Ç–µ —Å —Ñ–æ—Ä–º–∞—Ç–∏—Ä—É—é—â–µ–π —Å—Ç—Ä–æ–∫–æ–π, –∫–æ—Ç–æ—Ä–∞—è —Å–æ–¥–µ—Ä–∂–∏—Ç –æ–±—ã—á–Ω—ã–π —Ç–µ–∫—Å—Ç —Å–æ "—Å–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ç–æ—Ä–∞–º–∏ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤" - —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–º–∏ —Å–∏–º–≤–æ–ª–∞–º–∏, —Ç–∞–∫–∏–º–∏ –∫–∞–∫ "%s", "%d", "%f", "<small>number of digits</small>".

- %s - –°—Ç—Ä–æ–∫–∞ (–∏–ª–∏ –ª—é–±–æ–π –æ–±—ä–µ–∫—Ç —Å–æ —Å—Ç—Ä–æ–∫–æ–≤—ã–º –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ–º, –Ω–∞–ø—Ä–∏–º–µ—Ä, —á–∏—Å–ª–∞)
- %d - –¶–µ–ª—ã–µ —á–∏—Å–ª–∞
- %f - –ß–∏—Å–ª–∞ —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π
- "%.<small>number of digits</small>f" - –ß–∏—Å–ª–∞ —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π —Å —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–π —Ç–æ—á–Ω–æ—Å—Ç—å—é

```py

# –°—Ç—Ä–æ–∫–∏ –∏ —á–∏—Å–ª–∞

pi = 3.14
```

### –ú–µ—Ç–æ–¥—ã —Å—Ç—Ä–æ–∫

- count(): –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤—Ö–æ–∂–¥–µ–Ω–∏–π –ø–æ–¥—Å—Ç—Ä–æ–∫–∏ –≤ —Å—Ç—Ä–æ–∫—É. –ú–µ—Ç–æ–¥ –∏–º–µ–µ—Ç —Å–ª–µ–¥—É—é—â–∏–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å: count(substring, start=..., end=...). –ê—Ä–≥—É–º–µ–Ω—Ç start –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–∞—á–∞–ª—å–Ω—ã–π –∏–Ω–¥–µ–∫—Å, –∞ –∞—Ä–≥—É–º–µ–Ω—Ç end –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –ø–æ—Å–ª–µ–¥–Ω–∏–π –∏–Ω–¥–µ–∫—Å, –≥–¥–µ –±—É–¥–µ—Ç –≤–µ—Å—Ç–∏—Å—å –ø–æ–∏—Å–∫ –ø–æ–¥—Å—Ç—Ä–æ–∫–∏.

   –ü–æ–¥—Ä–æ–±–Ω–µ–µ –æ —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–∏ —Å—Ç—Ä–æ–∫ –º–æ–∂–Ω–æ —É–∑–Ω–∞—Ç—å –ø–æ [—Å—Å—ã–ª–∫–µ](https://www.programiz.com/python-programming/methods/string/format).
